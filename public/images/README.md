# Carpeta de imágenes de productos

Aquí se deben colocar las imágenes reales de los productos cuando estén disponibles.

- Durante el desarrollo, se usará un placeholder (`/placeholder.jpg`) para todos los productos que no tengan imagen real.
- Cuando se implemente el uploader, las imágenes reales se guardarán aquí y se referenciarán desde la base de datos.

**Ejemplo de ruta de imagen real:**
```
/images/mi-planta-bonita.jpg
```

**Placeholder temporal:**
```
/placeholder.jpg
```

Puedes cargar aquí las imágenes reales cuando estén listas.
